# 备份与还原 (Backup)

进入「设置」->「备份与还原」，加密导出/导入应用配置、聊天历史与凭据。

## 1. 导出备份
*   **选择导出数据**：勾选要导出的数据范围，包括 AI 提供商、Git 凭据、远程连接与挂载、聊天历史、MCP 服务器、授权规则、应用设置。
*   **设置导出口令**：留空则输出明文 tar.gz；填写则用口令加密（AES-GCM）。
*   导出格式为 tar.gz 压缩包，含 API Key 等凭据，通过系统文件选择器保存到指定位置。

## 2. 导入备份
*   从备份文件还原数据，合并方式导入，不清除现有数据。
*   需输入对应口令；备份未加密则留空。
*   导入完成后展示已还原数据汇总（提供商、Git 凭据、远程连接、远程挂载、聊天会话、聊天消息、待办项、MCP 服务器、全局授权规则等数量）。

## 3. 注意事项
*   口令丢失则加密备份无法恢复。
*   导入后应用内仍按现状明文存储。
