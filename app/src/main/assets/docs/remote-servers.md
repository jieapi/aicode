# 远程工作区 (Remote Servers) 与同步设置

让手机通过 SFTP / FTP 连接远程云端服务器作为项目工作区。

## 1. 连接配置 (Tab 1)
点击“+”填写云主机连接参数：
*   **基本信息**：名称、协议 (SFTP / FTP)。
*   **连接信息**：主机 IP 地址、端口（SFTP 默认 22，FTP 默认 21）。
*   **认证信息**：用户名和密码（或私钥）。配置完成可随时点击测试连接。

## 2. 工作区挂载映射 (Tab 2)
点击“+”选择一个已建好的连接，建立目录映射关系：
*   **本地目录**：Android 手机本地的工作区文件夹。
*   **远程目录**：服务端对应的开发目录。
*   在映射列表中可以随时点击按钮执行上传、下载或开启实时连接同步。

## 3. WiFi 传输服务 (Tab 3) - 内置 FTP 服务端
手机端内置轻量级 FTP 服务器，方便电脑端远程管理手机项目：
*   **启动服务**：开启开关后，会显示当前的访问连接（如 `ftp://192.168.1.100:2121`）。
*   **连接方式**：在电脑（Windows/Mac）的文件管理器或 FileZilla 地址栏中输入该连接。
*   **权限管理**：支持修改账号密码、端口号，以及开启“允许匿名访问”和“启动时自动开启”。
*   **共享目录**：连接后即可访问并管理手机上 `projects/` 目录下的所有开发工作区。

## 4. 同步规则与设置 (右上角齿轮)
可配置底层文件同步引擎的行为：
*   **忽略文件规则 (Ignored Patterns)**：配置过滤不需要同步的文件（如 `build/`、`.gradle/` 等）。
*   **开启 GitIgnore**：开关，决定是否读取项目中的 `.gitignore` 进行同步过滤。
*   **最大同步并发量**：调整传输时的线程并发数。
