# AI 代码编辑器 - UI 设计文档

## 1. 设计规范

### 1.1 颜色方案

**深色主题 (默认)**
```
主背景: #1E1E1E
侧边栏: #252526
编辑器: #1E1E1E
状态栏: #007ACC
文字: #D4D4D4
高亮: #264F78
边框: #3C3C3C
```

**浅色主题**
```
主背景: #FFFFFF
侧边栏: #F3F3F3
编辑器: #FFFFFF
状态栏: #007ACC
文字: #333333
高亮: #ADD6FF
边框: #E0E0E0
```

### 1.2 字体

```
代码字体: JetBrains Mono, Fira Code, Source Code Pro
UI 字体: Roboto, Noto Sans SC
字号: 14sp (代码), 16sp (UI)
```

### 1.3 图标

使用 Material Icons，主要图标：
- 文件类型: `insert_drive_file`, `folder`, `folder_open`
- 操作: `add`, `delete`, `edit`, `save`, `play_arrow`
- 导航: `arrow_back`, `menu`, `more_vert`
- AI: `auto_awesome`, `smart_toy`, `psychology`

```
┌─────────────────────────────────────┐

│ 🔤 complete                       │  ← 本地补全
│ 📝 completion                     │
│    completeWith()                   │
│    completionHandler()              │

- 使用不同图标区分补全类型
- AI 补全显示闪电图标
- 按 Tab 或点击接受补全
- 按 Esc 关闭弹窗
```

## 5. 动画与过渡

### 5.1 页面过渡
- 使用 `AnimatedContent` 实现页面切换
- 持续时间: 300ms
- 缓动曲线: `FastOutSlowInEasing`

### 5.2 组件动画
- 补全弹窗: 从下往上滑入 (200ms)
- 文件树展开: 渐显 + 缩放 (150ms)
- 标签切换: 淡入淡出 (200ms)
- 侧边栏: 从左滑入 (250ms)

### 5.3 反馈动画
- 按钮点击: 缩放 (0.95) + 波纹效果
- 长按: 震动反馈 + 弹出菜单
- 拖拽: 跟随手指 + 阴影效果

## 6. 响应式设计

### 6.1 断点
```
手机竖屏: < 600dp
手机横屏: 600dp - 840dp
平板: > 840dp
```

### 6.2 布局适配
- **竖屏**: 文件树作为抽屉，编辑器全屏
- **横屏**: 文件树在左侧 (240dp)，编辑器在右侧
- **平板**: 文件树在左侧 (300dp)，编辑器在右侧，可拖拽调整宽度

## 7. 手势操作

### 7.1 编辑器
- 双指缩放: 调整字体大小
- 左右滑动: 水平滚动
- 上下滑动: 垂直滚动
- 双击: 选中单词
- 三击: 选中整行

### 7.2 文件树
- 点击: 选择/展开
- 长按: 上下文菜单
- 左滑: 删除
- 右滑: 重命名

### 7.3 标签栏
- 点击: 切换标签
- 左滑: 关闭标签
- 长按: 操作菜单
